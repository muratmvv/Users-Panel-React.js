# React.js Tabanlı Kullanıcı Yönetim Paneli
Bu proje, modern web uygulamaları için ihtiyaç duyulan kullanıcı verilerinin yönetimi, görüntülenmesi ve düzenlenmesi süreçlerini kapsayan, modüler ve yüksek performanslı bir yönetim arayüzü (Admin Dashboard) çalışmasıdır. React.js kütüphanesinin bileşen tabanlı mimarisi kullanılarak geliştirilmiştir.

Teknik Mimari ve Özellikler

Uygulama, kullanıcı deneyimini ve veri yönetimini optimize etmek adına aşağıdaki teknik özellikleri barındırmaktadır:

Bileşen Tabanlı Yapı: Arayüz, yeniden kullanılabilir ve sürdürülebilir React bileşenleri ile inşa edilmiştir.

State Yönetimi: Kullanıcı verileri ve arayüz durumları (yükleme ekranları, hata yönetimi, form durumları) React Hooks (useState, useEffect) kullanılarak yönetilmiştir.

Dinamik Veri Listeleme: API veya yerel veri setlerinden gelen kullanıcı bilgileri, yüksek performanslı liste bileşenleri ile render edilmektedir.

Form Yönetimi ve Validasyon: Yeni kullanıcı ekleme ve mevcut verileri güncelleme işlemleri için optimize edilmiş form yapıları kullanılmıştır.

Responsive Tasarım: Arayüz, farklı ekran boyutlarında sorunsuz çalışması için esnek (Grid/Flexbox) tasarım prensipleriyle geliştirilmiştir.

Kullanılan Teknolojiler

React.js: UI bileşenleri ve uygulama mantığı.

JavaScript (ES6+): Dinamik veri işleme ve modern yazım standartları.

CSS3 / Styled Components: Görsel tasarım ve kullanıcı arayüzü stili.

React Router: Sayfa içi navigasyon ve kullanıcı detay sayfaları.

Kurulum ve Başlatma

Projenin yerel ortamınızda çalışması için bağımlılıkların yüklenmesi ve geliştirme sunucusunun başlatılması gerekmektedir:

Bağımlılıkları yükleyin: npm install

Uygulamayı başlatın: npm start
